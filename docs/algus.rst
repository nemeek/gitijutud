Algus
=====

Gitiga alustamiseks on kaks võimalust: kas kloonida olemasolev projekt
või luua uus varamu oma arvutisse.

Uue projekti algus
++++++++++++++++++

``git init``

.. sidebar:: Giti installeerimine

	     Linuxis ja Macis on git vaikimisi installeeritud.


Kloonimine
++++++++++

``git clone``
